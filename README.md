# 🚀React Hooks + Next.js + Egg.js + MySql + GraphQL + Ant Design 博客系统

## 技术栈

* React
* Next
* TypeScript
* Webpack
* Node.js
* Egg
* TSLint

## blog/admin/service

### 项目拉取后需要进入相应目录分别进行安装依赖项

以bolg前台为例：

```bash
cd blog
yarn install
```

## blog

### 项目启动

```bash
yarn dev
```

## admin

### 项目启动（开发模式）

```bash
yarn start
```

### 项目打包

由于是个人后台管理页面放到其他端口或域名以静态页面模式使用

```bash
yarn build
```

## service

### 项目启动（开发模式）

```bash
npm i
npm run dev
open http://localhost:7001/
```

### 项目部署

```bash
npm start
npm stop
```
